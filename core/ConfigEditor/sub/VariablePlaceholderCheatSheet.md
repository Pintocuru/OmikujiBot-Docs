# 評価ブロック (変数プレースホルダー) チートシート

## 📌 クイックリファレンス

| 機能           | 書式例                                    | 説明                |
| ------------ | -------------------------------------- | ----------------- |
| **文字列代入**    | `{{name = 'さくら'}}`                     | シングルクォート必須        |
| **数値代入**     | `{{score = 100}}`                      | そのまま数値            |
| **変数コピー**    | `{{copy = score}}`                     | score の値をコピー      |
| **加算**       | `{{score = score + 5}}`                | score に 5 を加算     |
| **減算**       | `{{hp = hp - 1}}`                      | hp から 1 を減算       |
| **乗算**       | `{{attack = attack * 2}}`              | attack を 2 倍      |
| **除算**       | `{{total = total / 2}}`                | total を 2 で割る     |
| **三項演算子**    | `{{result = score >= 50 ? 'A' : 'B'}}` | 条件分岐              |
| **ランダム**     | `{{dice = rand(1, 6)}}`                | 1〜6 の乱数を代入        |
| **文字列化**     | `{{text = str('1/', val)}}`            | 明示的な文字列連結         |
| **数値化**      | `{{numValue = num(textValue)}}`        | 文字列を数値に変換（失敗時 0）  |
| **真偽値化**     | `{{flag = bool(score)}}`               | 0 以外を 1、0 を 0 に変換 |
| **複数式**      | `{{a = 1; b = 2; c = a + b}}`          | セミコロン区切り          |

## 出力方法

| 機能           | 書式例                                    | 説明                |
| ------------ | -------------------------------------- | ----------------- |
| **変数参照（出力）** | `[[score]]`                            | 値をテキストに出力         |

---

## 📝 基本：代入と参照

代入タグはテキストに**何も出力しません**。値を表示するには参照タグを使います。

```javascript
{{score = 100}}           // 代入（出力なし）
スコアは [[score]] 点！   // 参照（出力あり）
// → スコアは 100 点！
```

### 値の種類

```javascript
{{name = 'さくら'}}    // 文字列: シングルクォート必須
{{level = 50}}          // 数値: クォート不要
{{copy = level}}        // 変数コピー: level の値が入る（未定義なら 0）
```

---

## 🧮 四則演算

```javascript
{{score = 0}}

{{score = score + 10}}   // score = 10
{{score = score - 3}}    // score = 7
{{score = score * 2}}    // score = 14
{{score = score / 2}}    // score = 7
```

**数値範囲**: -999,999,999 ～ 999,999,999（超過時は上限/下限に丸め） **小数点**: 0.01 単位まで有効

---

## 🔀 三項演算子（条件分岐）

```javascript
// 書式: {{変数 = 条件 ? 真の値 : 偽の値}}

{{score = score + 10}}
{{result = score >= 80 ? 'S' : score >= 50 ? 'A' : 'B'}}
[[result]]
// score が 80 以上なら S、50 以上なら A、それ以外なら B
```

使用できる比較演算子: `==` `!=` `>` `<` `>=` `<=`（数値比較のみ）

---

## 🎲 rand（ランダム）

```javascript
{{dice = rand(1, 6)}}       // 1〜6 の乱数を代入
{{score += rand(1, 10)}}    // 1〜10 の乱数を加算
[[dice]]                    // 結果を出力
```

---

## ⚡ 複数式（セミコロン区切り）

1 つのタグに複数の式をまとめて書けます。

```javascript
{{score = 0; hp = 100; name = 'hero'}}

// 初期化をまとめて書く場合などに便利
{{base = 50; bonus = rand(1, 20); total = base + bonus}}
[[total]]
```

---

## ⚙️ 応用パターン

### プレースホルダーとの連携

```javascript
// ユーザーIDを変数に保存
{{userId = <<ユーザーID>>}}

// 動的な変数名
{{<<アイテム名>> = <<価格>>}}
// <<アイテム名>> が「sword」→ {{sword = 100}} として実行
```

### ランキングシステム

```javascript
{{score += rand(1, 100)}}
{{text = score}}     // 予約語「text」にスコアをセット
[[score]]
```

### カウンター

```javascript
{{visitCount += 1}}
[[visitCount]] 回目の訪問！
```

### 条件によるメッセージ分岐

```javascript
{{score = score + rand(1, 100)}}
{{msg = score >= 80 ? '大吉！' : score >= 50 ? '中吉' : '小吉'}}
[[msg]]（[[score]] 点）
```

## ⚠️ 予約語リスト

| 予約語             | 用途例                | デフォルト   | 使用例                   |
| --------------- | ------------------ | ------- | --------------------- |
| `name`          | 名前/称号              | (ユーザー名) | `name = 'ドラちゃん'`      |
| `symbol`        | シンボル/アイコン          |         | `symbol = 'bar'`      |
| `text`          | 表示テキスト             |         | `text = 'foo'`        |
| `order`         | 順番 (ランキング等での比較値)    | 0       | `order = 'bar'`       |
| `variant`       | カラー指定（DaisyUI カラー） | primary | `variant = 'success'` |
| `unique`        | 同じユーザーならまとめるか      | false   | `unique = 'true'`     |
| `visible`       | 可視化するか             | true    | `visible = 'false'`   |
| `flag`          | 何らかのフラグ            | false   | `flag = 'true'`       |
| `slot0`〜`slot9` | 追加データスロット          |         | `slot0 = 'buz'`       |

### variant カラー

| カラー名        | 色合いのイメージ    | 使用例                     |
| ----------- | ----------- | ----------------------- |
| `none`      | なし          | `variant = 'none'`      |
| `primary`   | ブルー系（テーマ依存） | `variant = 'primary'`   |
| `secondary` | パープル/グレー系   | `variant = 'secondary'` |
| `accent`    | ピンク/シアン系    | `variant = 'accent'`    |
| `info`      | シアン/ライトブルー  | `variant = 'info'`      |
| `success`   | グリーン        | `variant = 'success'`   |
| `warning`   | イエロー/オレンジ   | `variant = 'warning'`   |
| `error`     | レッド         | `variant = 'error'`     |
| `neutral`   | グレー系        | `variant = 'neutral'`   |

## 🚫 制限事項

1. **数値範囲**: -999,999,999 ～ 999,999,999
2. **小数点**: 0.01 単位まで（以下は丸め）
3. **変数名**: 英数字のみ、先頭は英字
4. **文字列**: シングルクォートで囲む（ダブルクォート不可）
5. **文字列比較**: 三項演算子での文字列比較は不可（数値のみ）
6. **処理順**: プレースホルダー → 変数プレースホルダー

---

## 📦 旧構文（非推奨、v3 で廃止）

後方互換のため動作しますが、新しく書く場合は新構文を推奨します。

| 旧構文                      | 新構文                       |
| ------------------------ | ------------------------- |
| `{{var name = 'value'}}` | `{{name = 'value'}}`      |
| `{{rand(1, 10)}}`        | `{{r = rand(1,10)}}[[r]]` |
