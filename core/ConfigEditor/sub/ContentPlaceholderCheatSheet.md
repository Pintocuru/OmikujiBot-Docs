# プレースホルダー チートシート

## 📋 クイックリファレンス

### 構文一覧

| 種類 | 構文例 | 説明 |
|------|--------|------|
| インラインランダム | `<<'A','B','C'>>` | 即席でランダム選択 |
| 定義済み参照 | `<<天気>>` | 事前設定した値を参照 |
| 予約語（共通） | `<<viewer>>` | 同時接続視聴者数 |
| 予約語（コメント専用） | `<<user>>` | ユーザー名 |

### 予約語早見表

| 用途 | 予約語 | 取得する情報 |
|------|--------|--------------|
| **配信統計** | `<<viewer>>` | 同時接続数 |
| | `<<upVote>>` | 高評価数 |
| | `<<commenter>>` | 累計コメントユーザー数 |
| **ユーザー情報** | `<<user>>` | ユーザー名/ニックネーム |
| | `<<userId>>` | ユーザーID |
| | `<<tc>>` | そのユーザーの総コメント数 |
| **その他** | `<<winner>>` | ランダム抽選ユーザー |
| | `<<price>>` | ギフト金額 |
| | `<<lc>>` | 配信全体のコメント通し番号 |
| | `<<draws>>` | おみくじ引き回数 |
