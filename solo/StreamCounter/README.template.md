## このテンプレートは何？（Features）

![](images/Pasted%20image%2020260313114941.png)

### 高評価も！コメント数も！いろいろカウントできる多機能カウンターテンプレート

- わんコメと連携して動作する【おみくじ BOT】を利用した、配信の気になる数値を表示するカウンターテンプレートです。
- ライブ配信では気になる、高評価・同時接続数はもちろん、チャットのコメント数や、コメントを残したユーザー数も数えてくれます
	- 初見さん、ユーザー数、高評価、同時接続数、コメント数、ギフト金額 (配信の合計)、フォロワー数 7 つです

### 特定のワードのカウントもできる

- さらに、チャットから「特定のワード」を指定することで、そのカウントも行ってくれます。
- 「おみくじ BOT」の設定を利用することで、ユーザーごとのカウント、1 回で複数増加することも可能。
- おはようカウンター、ワロタカウンター、ナイスカウンターを搭載した無料版をご覧ください。

### カスタマイズすごい

- 「ライバーカウンター」がこだわっているのは、エディターのしやすさ
- 「おみくじ BOT」のエディターにより前バージョンより更にエディットがしやすくなりました。
- 無料版でも 8 色、有料の Adv 版なら 35 テーマのカラーバリエーション
- PRO 版はフォント変更、UI 変更もできる！

### シーン別・活用例

- **朝活配信**
  - 画面の隅に設置することで、画面が賑やかになります。
  - コメント数やギフト金額が表示され、賑わいがわかります
- 雑談配信
	- ユーザーが増えるたびにリストが埋まっていき、賑わいが可視化されます。
	- 並び方で話が弾むかも。
- 抽選・参加確認・ちょっとした遊びに最適
	- 「誰が参加したか」「どこに座るか」を視覚的に楽しめるため、 抽選演出やアイスブレイク用途に向いています。
